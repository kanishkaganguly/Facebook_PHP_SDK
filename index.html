Facebook PHP SDK

Welcome to Facebook's PHP SDK.

This webpage contains the resources needed to learn and understand the Facebook PHP SDK and the OpenGraph. Since I am a beginner myself, it is easy for me to present the necessary tutorials in a very simple and lucid form.

Basic Instructions

Since this is a local project, a certain setup is needed.

1. Download [WAMPServer](http://www.wampserver.com/en/) or [MAMPServer](http://www.mamp.info/en/index.html) and      install it on your computer. 
2. Go to the /www/ folder in your system root.
3. Create a folder "GraphAPI" and in it create folder "Facebook_PHP_SDK"
4. Place all contents of zip file into this folder.
5. Go to http://localhost/GraphAPI/Facebook_PHP_SDK/index.php on your browser.
